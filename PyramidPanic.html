Pyramid Panic

Pyramid Panic is a multiplayer survival royal game where players spawn inside a pyramid tomb and need to quickly find the exit. Along the way players will find weapons like swords, bows, shields, and armor to combat other players and enemy A.I. bugs. The map is procedurally generated every round creating a unique experience. The game was built using C++ from a game engine made in the previous semester.
